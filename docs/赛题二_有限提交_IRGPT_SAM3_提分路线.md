## 每日 4 次提交策略

每天最多安排：

1. **S1 稳健确认或回退基线**：只在平台状态、打包方式或昨天最佳版本需要复核时使用。
2. **S2 高置信单变量消融**：例如只调整存在性阈值。
3. **S3 第二个独立单变量消融**：例如只调整 prompt normalization 或 mask 阈值。
4. **S4 探索版**：例如 IRGPT 伪标签、小目标 crop、mask 后处理；如果 S2/S3 有明显收益，则改为收益叠加验证。

任何低于当前最佳线上分数的版本立即归档，不继续叠加改动。

## IRGPT 使用方向

IRGPT 建议作为红外语义先验生成器，而不是直接替代分割模型：

- 为无标注红外图生成场景类型、候选类别、实例粗框、置信度、热异常描述和小目标候选。
- 使用 IRGPT 的 box/text prompt 引导 SAM3 生成像素级 mask。
- 保存结构化伪标签字段：、、、、、、、。
- 第一阶段只覆盖高价值类别：person、car/vehicle、road、building、tree、pole、power line、insulator、animal。

训练时按质量分数加权，低质量标签只做弱监督或暂不进入训练集。
